LeedAB is the AI operating layer for supply chain and logistics teams. It runs as a local agent system — the **Brain** — inside your office, on hardware you already own. Five **Artificial Beings** ship out of the box. Each one owns a specific procurement loop, with memory, monitoring, and a paper trail.

No API integrations. No cloud uploads. Your data does not leave the room.

## How it works

Live in days, not months.

<Steps>
  <Step title="Install · Day 0">
    An appliance ships to your office, or runs on hardware you already own — any spare Mac, Windows PC, or Linux box. No cloud account to provision.

    ```bash macOS / Linux
    curl -fsSL https://leedab.com/install.sh | bash
    ```

    ```powershell Windows
    powershell -c "irm https://leedab.com/install.ps1 | iex"
    ```

    ```bash npm
    npm install -g leedab
    ```
  </Step>
  <Step title="Connect · Days 1–3">
    Sign in once, like a person. A LeedAB engineer pairs with your team to log the Brain into SAP, carrier portals, TMS dashboards, your inbox, and any supplier portals that matter. No API contracts. No IT integration tickets.
  </Step>
  <Step title="Work · Days 4–14">
    Five ABs wake up. The control tower starts reconciling allocation. The screener starts pre-flighting orders. The follow-up agent starts chasing commits. Your team meets them in Slack, Teams, WhatsApp, or Telegram — as colleagues.
  </Step>
</Steps>
